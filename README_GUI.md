# BIAS-ATR 智能网格交易系统 - GUI版

一个基于tkinter的现代化桌面GUI应用，提供ETF网格交易监控和管理的完整解决方案。

## 🎯 主要功能

### 1. 实时ETF监控面板
- 📊 实时显示ETF价格、BIAS指标、市场状态
- 🎯 智能网格买卖建议（买1、买2、卖1、卖2等）
- 📈 持仓信息和市值计算
- 🔄 自动数据刷新（可配置间隔）

### 2. 价格提醒系统
- 🔔 实时价格提醒，当价格触及网格买卖点时自动通知
- 💰 多样化提醒方式：弹窗通知、浏览器通知、声音提醒
- 📝 提醒历史记录和统计
- 🎛️ 灵活的提醒设置和过滤

### 3. 交易功能界面
- 💼 手动下单对话框
- 📋 交易记录和历史查询
- ⚡ 快速交易操作
- 🔒 风险控制和限制设置

### 4. 数据分析和可视化
- 📊 价格走势图表
- 📈 BIAS指标分析
- 💰 盈亏分布统计
- 📋 数据导出功能

### 5. 系统设置和管理
- ⚙️ 全面的系统配置选项
- 🎛️ 策略参数调整
- 🔔 提醒设置配置
- 💾 数据备份和恢复

## 🚀 快速开始

### 环境要求
- Python 3.8+
- tkinter (通常随Python一起安装)
- pandas, numpy
- matplotlib (用于数据分析)
- tushare 或 akshare (数据源)

### 安装依赖
```bash
pip install pandas numpy matplotlib tushare akshare
```

### 启动程序

#### 演示版（推荐首次使用）
```bash
python run_gui.py
```

#### 完整版（需要配置数据源）
```bash
python gui_main.py
```

### 关键区域说明

#### 1. 顶部信息栏
- **系统标题**：显示系统名称和模式
- **数据源状态**：显示当前数据源类型
- **更新时间**：显示最后更新时间
- **快捷按钮**：刷新、设置、提醒等功能

#### 2. 汇总卡片
- **总资产**：配置的总资金额
- **持仓市值**：当前ETF持仓总市值
- **可用现金**：剩余可用资金
- **总盈亏**：总体盈亏金额
- **仓位比例**：当前仓位占总资产比例

#### 3. ETF监控表格
- **代码/名称**：ETF基本信息
- **价格/BIAS**：实时价格和BIAS指标
- **状态**：根据BIAS值的市场状态
- **仓位/市值**：持仓信息和市值
- **订单**：网格建议订单（绿色=买入，红色=卖出）
- **提醒**：价格提醒数量

#### 4. 右侧控制面板
- **价格提醒**：实时提醒信息显示
- **快速操作**：常用功能快捷按钮
- **系统状态**：系统运行状态信息

## 🎛️ 使用指南

### 基本操作流程

1. **启动监控**
   - 运行程序后自动开始监控配置的ETF
   - 系统会定期获取最新价格和计算网格建议

2. **查看网格建议**
   - 在ETF表格中查看建议订单
   - 🔵 绿色表示买入建议
   - 🔴 红色表示卖出建议

3. **价格提醒**
   - 当价格触及网格买卖点时自动提醒
   - 可在右侧面板查看提醒详情
   - 支持系统通知（需授权）

4. **手动交易**
   - 双击ETF行查看详情
   - 使用"手动下单"功能进行交易
   - 支持使用网格建议价格

### 高级功能

#### 策略配置
- 通过"设置"菜单调整BIAS阈值
- 配置目标仓位比例
- 设置风控参数

#### 数据分析
- 使用"数据分析"功能查看图表
- 导出历史数据进行回测
- 监控策略表现

#### 提醒管理
- 配置提醒方式和频率
- 查看提醒历史记录
- 设置提醒过滤条件

## ⚙️ 配置说明

### 数据源配置
支持多种数据源：
- **akshare**（推荐）：免费、稳定、数据丰富
- **tushare**：需要注册token，功能强大

### 策略参数
- **BIAS阈值**：定义市场状态的分界点
- **目标仓位**：各状态下的目标持仓比例
- **网格间距**：基于ATR的动态间距计算

### 提醒设置
- **提醒方式**：弹窗、系统通知、声音
- **提醒频率**：避免重复提醒的时间间隔
- **过滤条件**：按ETF、价格范围等过滤

## 🔧 故障排除

### 常见问题

1. **GUI启动失败**
   - 检查Python版本（需要3.8+）
   - 确认tkinter已正确安装
   - 检查依赖包是否完整

2. **数据获取失败**
   - 检查网络连接
   - 确认数据源配置正确
   - 查看API限制和配额

3. **提醒不工作**
   - 检查系统通知权限
   - 确认提醒设置已启用
   - 检查价格是否确实触及点位

### 调试模式
```bash
# 启用详细日志
python run_gui.py --debug

# 查看配置信息
python run_gui.py --show-config
```
